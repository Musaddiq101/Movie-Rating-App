# Movie Rating App

Movie Rating App allowing you to search for movies and siplay their information.
Using React and redux with Redux Toolkit for state management along with axios for api calls from omdbmovies api.

## Install Dependencies and Run

Install all the dependencies before starting the app using:

```bash
npm install
```

In the project directory, you can run the app using:

 ```bash
 npm start
 ```

Runs the app in the development mode.\
Open [http://localhost:3000](http://localhost:3000) to view it in your browser.
